Error 1000

Sorry, the connection to the data server was lost. As the session state is still valid, all the session data including device settings has been saved to a file with name 'last_session_[file-name]' where file-name is the last loaded settings file.

Cases that may trigger this failure

  • A user closed the data server application
  • The data server runs on a different computer and network connection was lost.
  • The data server could not handle an error and was closed.
