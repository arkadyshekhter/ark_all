Zurich Instruments LabOne Instrument Driver Readme

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Zurich Instruments, http://www.zhinst.com
Supported Language: LabVIEW
Supported Models: MFLI, HF2LI, HF2IS, UHFLI
Models Tested: MFLI, HF2LI, HF2IS, UHFLI
Interfaces: USB, TCP/IP, all via ziServer
Firmware Revisions Tested: 15.11

Certified: No
NI Supported: No
Source Code Available: No

Driver Revision: 15.11
Original Release Date: 08/04/2009
Current Revision Date: 21/12/2015

2. Required Software

Install LabOne prior to use of this LabVIEW driver.

LabOne v15.11 or later

Windows LabVIEW 2009 or later, x86 and 64-bit version
Mac OSX LabVIEW 2009 or later, x86 and 64-bit version
Linux LabVIEW 2010 or later, x86 version


Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

3. Known Issues

To report issues or provide feedback about this instrument driver, please send an email to support@zhinst.com.

4. Revision History

The latest version of this and other LabVIEW and LabWindows/CVI instrument drivers can be downloaded at the Instrument Driver Network.

v15.11, 2015-11-20

Added ziCore support for Software Trigger.
Added support for Mac OSX Labview 64.

v15.05, 2015-06-20

Added support for MFLI.
Added ziCore support for Sweeper and Spectrum.
Added support for Mac OSX Labview x86.
The ziSetValue* vi's now also accept arrays of paths and values.
Added vi for determing hostname and port for ziOpen.vi.

v14.08, 2014-09-24

Improved examples and improved error messages of API.
Fixed ziGetEventByteArray.vi, which didn't work on UHFLI.

v14.02, 2014-04-24

Renamed "Zurich Instruments HF2" to "Zurich Instruments LabOne". This implies adaptations in your LabVIEW programs. Upon loading your program you need to manually direct LabVIEW to find the vi's at the new location. Then save your vi.
Added support of timestamps on polls of double and integer values for UHFLI.
Improved support for UHFLI.
Various fixes.

v12.08.2, 2013-08-08

Min requirement on LabVIEW version is up from 8.2 to 2009.
Added support for 64-bit Windows version of LabVIEW.
Improved support for UHFLI.
Various fixes.

v12.02.0, 2012-08-08

Improved ZoomFFT.
Improved impedance functions.

v11.02.1, 2011-04-20

Fixed ZoomFFT.
Added ziFlush.vi, ziSweepPtbyPt.vi, ziAutoRange.vi.
Reworked impedance functions.

v10.03.1, 2010-07-20

Added ziTransferFunction.vi, ZoomFFT.vi, ZoomFFT-SNR.vi and HF2CA-Impedance.vi.

v10.03, 2010-03-30

Added ziBW2TC and ziTC2BW.
Changed LoadCfg to not abort on unknown nodes.
Fixed bug in poll of DIO values.
Removed "Channel" field in output of ziGetDemodSample.vi.
Changed format of DIOInput and AuxInput in ziPollData.vi and ziTypePollData.ctl.

v9.11, 2009-12-08

Added Sweeper vi's and Examples.
